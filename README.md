# Django About Russia

Учебный web-проект на Django, выполненный в рамках обучения.  
Проект представляет собой простой многостраничный сайт и используется для отработки базовых возможностей фреймворка.

## Функциональность
- Многостраничная структура сайта  
- Работа с шаблонами Django  
- Использование моделей и базы данных  
- Административная панель  

## Технологии
- Python  
- Django 5.x  
- Django ORM  
- SQLite  
- HTML / CSS  

## Цель проекта
Закрепление базовых навыков работы с Django: структура проекта, шаблоны, модели, маршрутизация.
